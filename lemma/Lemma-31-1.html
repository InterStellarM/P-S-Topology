Lemma 31.1

Let \(X\) be a topological space. Let one-point sets in \(X\) be closed.

  1. (a)\(X\) is regular if and only if given a point \(x\) of \(X\) and a neighborhood \(U\) of \(x\), there is a neighborhood \(V\) of \(x\) such that \(x \in U \subseteq\overline{V} \subset U\).
  2. (b)\(X\) is normal if and only if given a closed set \(A\) and an open set \(U\) containing \(A\), there is an open set \(V\) containing \(A\) such that \(\overline{V} \subset U\).
